# GeothermalProperties webapp

The GeothermalProperties webapp serves as a user interface to manage data associated with GeothermalProperties microservice

It is packaged as a docker container called:

``norcedrillinggeothermalpropertieswebappclient``

It is available on dockerhub, under the digiwells organization, at:

https://hub.docker.com/?namespace=digiwells

The API (OpenApi schema) of the microservice is available and testable at:

https://dev.digiwells.no/GeothermalProperties/api/swagger (development server) 

https://app.digiwells.no/GeothermalProperties/api/swagger (production server)

The webapp itself is available at:

https://dev.digiwells.no/GeothermalProperties/webapp/GeothermalPropertiesCompletionOrder

https://app.digiwells.no/GeothermalProperties/webapp/GeothermalPropertiesCompletionOrder
